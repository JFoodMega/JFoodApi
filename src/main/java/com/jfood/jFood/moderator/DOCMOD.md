# Moderator API

**Base URL:** `http://localhost:8080`

---

## Эндпоинты для админа

> ⚠️ Создание, удаление модераторов — только для администратора.

---

### 1. Получить список модераторов

```
GET /moderators/admin
```

**Ответ `200`:**

```json
[
  {
    "id": 1,
    "name": "Иван Петров",
    "phone": "+79001234567"
  }
]
```

---

### 2. Создать модератора

```
POST /moderators/admin
```

**Тело запроса:**

```json
{
  "login": "moderator1",
  "password": "secret123",
  "phone": "+79001234567",
  "name": "Иван Петров"
}
```

**Обязательные поля:** `login`, `password`, `phone`

**Ответ `201`:**

```json
{
  "id": 1,
  "name": "Иван Петров",
  "phone": "+79001234567"
}
```

---

### 3. Удалить модератора

```
DELETE /moderators/admin/{moderatorId}
```

**Параметры пути:**

| Параметр | Тип | Описание |
|---|---|---|
| `moderatorId` | long | ID модератора |

**Ответ `204`** — модератор удалён, тело пустое

**Ответ `404`:**

```json
"Модератор не найден: 5"
```

---

## Эндпоинты для модератора

---

### 4. Получить модератора по ID

```
GET /moderators/{moderatorId}
```

**Параметры пути:**

| Параметр | Тип | Описание |
|---|---|---|
| `moderatorId` | long | ID модератора |

**Ответ `200`:**

```json
{
  "id": 1,
  "name": "Иван Петров",
  "phone": "+79001234567"
}
```

**Ответ `404`:**

```json
"Модератор не найден: 5"
```

---

### 5. Обновить профиль модератора

```
PATCH /moderators/{moderatorId}
```

**Параметры пути:**

| Параметр | Тип | Описание |
|---|---|---|
| `moderatorId` | long | ID модератора |

Все поля опциональные — передаёшь только те что меняешь.

**Тело запроса:**

```json
{
  "name": "Иван Сидоров",
  "phone": "+79009876543"
}
```

**Ответ `200`** — обновлённый объект модератора

**Ответ `404`:**

```json
"Модератор не найден: 5"
```

---

### 6. Получить новые заказы

```
GET /moderators/orders/new
```

Возвращает заказы со статусом `CREATED` — те что нужно назначить курьеру.

**Ответ `200`:**

```json
[
  {
    "id": 1,
    "clientId": 3,
    "courierId": null,
    "moderatorId": null,
    "status": "CREATED",
    "totalPrice": 850,
    "createdAt": "2026-04-24T10:00:00",
    "updatedAt": "2026-04-24T10:00:00",
    "assignedAt": null,
    "dishes": [
      {
        "id": 1,
        "name": "Борщ",
        "price": 350
      }
    ]
  }
]
```

---

### 7. Назначить курьера на заказ

```
POST /moderators/orders/{orderId}/assign
```

**Параметры пути:**

| Параметр | Тип | Описание |
|---|---|---|
| `orderId` | long | ID заказа |

**Тело запроса:**

```json
{
  "courierId": 2,
  "moderatorId": 1
}
```

**Ответ `200`** — обновлённый заказ со статусом `COURIER_ASSIGNED`

**Ответ `404`** — заказ, курьер или модератор не найден

**Ответ `400`:**

```json
"Заказ уже назначен или недоступен"
```

```json
"Курьер недоступен"
```

> ℹ️ После назначения курьер автоматически становится недоступным (`isAvailable: false`).
> Список доступных курьеров получай через `GET /couriers/available` перед назначением.

---

## Справочник статусов заказа

| Статус | Описание |
|---|---|
| `CREATED` | Новый заказ, ожидает назначения курьера |
| `COURIER_ASSIGNED` | Курьер назначен |
| `ON_THE_WAY` | Курьер в пути |
| `DELIVERED` | Доставлен |
| `CANCELED` | Отменён |
