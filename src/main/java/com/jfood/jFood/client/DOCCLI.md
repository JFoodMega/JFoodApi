# Client API

**Base URL:** `http://localhost:8080`

---

## Эндпоинты для клиента

### 1. Зарегистрировать клиента

```
POST /clients
```

**Тело запроса:**

```json
{
  "login": "user@example.com",
  "password": "secret123",
  "name": "Иван Петров",
  "phone": "+79001234567"
}
```

**Обязательные поля:** `login`, `password`, `name`, `phone`

**Ответ `201`:**

```json
{
  "id": 1,
  "login": "user@example.com",
  "name": "Иван Петров",
  "phone": "+79001234567",
  "addresses": []
}
```

---

### 2. Обновить профиль клиента

```
PATCH /clients/{clientId}
```

**Параметры пути:**

| Параметр   | Тип  | Описание   |
|------------|------|------------|
| `clientId` | long | ID клиента |

Все поля опциональные — передаёшь только те что меняешь.

**Тело запроса:**

```json
{
  "name": "Иван Сидоров",
  "phone": "+79009876543",
  "login": "new_login"
}
```

**Ответ `200`** — обновлённый профиль клиента

**Ответ `404`:**

```json
"Клиент с id=5 не найден"
```

---

### 3. Получить заказы клиента

```
GET /clients/{clientId}/orders
```

**Параметры пути:**

| Параметр   | Тип  | Описание   |
|------------|------|------------|
| `clientId` | long | ID клиента |

**Параметры пагинации (опциональные):**

```
?page=0&size=10&sort=createdAt,desc
```

**Ответ `200`:**

```json
{
  "content": [
    {
      "id": 1,
      "clientId": 1,
      "courierId": 2,
      "moderatorId": 1,
      "status": "DELIVERED",
      "totalPrice": 850,
      "createdAt": "2026-04-24T10:00:00",
      "updatedAt": "2026-04-24T12:00:00",
      "assignedAt": "2026-04-24T10:30:00",
      "dishes": [
        {
          "id": 1,
          "name": "Борщ",
          "price": 350
        }
      ]
    }
  ],
  "totalPages": 1,
  "totalElements": 1,
  "size": 10,
  "number": 0
}
```

## Адреса клиента

### 4. Получить адреса клиента

```
GET /clients/{clientId}/addresses
```

**Параметры пути:**

| Параметр   | Тип  | Описание   |
|------------|------|------------|
| `clientId` | long | ID клиента |

**Ответ `200`:**

```json
[
  {
    "id": 1,
    "city": "Москва",
    "street": "Ленина",
    "house": "10",
    "apartments": 5,
    "floor": 2,
    "entrance": 1
  }
]
```

---

### 5. Добавить адрес

```
POST /clients/{clientId}/addresses
```

**Параметры пути:**

| Параметр   | Тип  | Описание   |
|------------|------|------------|
| `clientId` | long | ID клиента |

**Тело запроса:**

```json
{
  "city": "Москва",
  "street": "Ленина",
  "house": "10",
  "apartments": 5,
  "floor": 2,
  "entrance": 1
}
```

**Обязательные поля:** `city`, `street`, `house`

**Ответ `201`** — созданный адрес

---

### 6. Обновить адрес

```
PATCH /clients/{clientId}/addresses/{addressId}
```

**Параметры пути:**

| Параметр    | Тип  | Описание   |
|-------------|------|------------|
| `clientId`  | long | ID клиента |
| `addressId` | long | ID адреса  |

Все поля опциональные.

**Тело запроса:**

```json
{
  "street": "Пушкина",
  "house": "15"
}
```

**Ответ `200`** — обновлённый адрес

**Ответ `404`** — клиент или адрес не найден

---

### 7. Удалить адрес

```
DELETE /clients/{clientId}/addresses/{addressId}
```

**Параметры пути:**

| Параметр    | Тип  | Описание   |
|-------------|------|------------|
| `clientId`  | long | ID клиента |
| `addressId` | long | ID адреса  |

**Ответ `204`** — адрес удалён, тело пустое

**Ответ `404`** — адрес не найден

## Эндпоинты для админа

> ⚠️ Эндпоинты ниже доступны только администратору.

---

### 8. Получить список клиентов

```
GET /clients/admin
```

**Параметры запроса (опциональные):**

| Параметр | Тип    | Описание                            |
|----------|--------|-------------------------------------|
| `search` | string | Поиск по имени, телефону или логину |
| `page`   | int    | Номер страницы (по умолчанию 0)     |
| `size`   | int    | Размер страницы (по умолчанию 20)   |

**Примеры:**

```
GET /clients/admin                        — все клиенты
GET /clients/admin?search=Иван            — поиск по имени
GET /clients/admin?search=+7900           — поиск по телефону
GET /clients/admin?search=user@           — поиск по логину
GET /clients/admin?page=1&size=10         — пагинация
```

**Ответ `200`:**

```json
{
  "content": [
    {
      "id": 1,
      "login": "user@example.com",
      "name": "Иван Петров",
      "phone": "+79001234567",
      "addresses": []
    }
  ],
  "totalPages": 1,
  "totalElements": 1,
  "size": 20,
  "number": 0
}
```

---

### 9. Получить клиента по ID

```
GET /clients/admin/{clientId}
```

**Параметры пути:**

**Ответ `200`** — объект клиента

**Ответ `404`:**

|------------|------|------------|

**Ответ `404`:**

```json
"Клиент с id=5 не найден"
```
